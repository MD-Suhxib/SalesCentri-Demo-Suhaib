Sanity Studio: `/studio`

Login: Use your Sanity.io account or SSO.

How to publish a blog post

1. Create → Post
2. Enter Title (~60–70 chars)
3. Set Slug (auto from title; can edit)
4. Add Excerpt (~150–160 chars)
5. Upload Main Image (add required Alt text)
6. Write content in Body (use H2/H3, lists, links, images, callouts)
7. SEO: set SEO Title/Description (optional), Canonical URL (optional)
8. Set Status → Published and ensure Published At is set
9. Click Publish

Preview draft

- Visit `/api/preview?secret=YOUR_SECRET&slug=your-post-slug`

Content standards

- Title: ~60–70 chars, clear keywords
- Meta description: ~150–160 chars
- Feature image: 1200×630, descriptive alt text
- Use H2/H3 headings, short paragraphs
- Use categories and author
